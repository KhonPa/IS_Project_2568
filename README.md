# IS_Project_2568
Project วิชา Intelligent System

คู่มือการใช้งานระบบพยากรณ์อัจฉริยะ (IS Project 2568)
Machine Learning (Ensemble) & Deep Learning (Neural Network)

1. ภาพรวมของโปรเจค (Project Overview)
โปรเจคนี้คือเว็บแอปพลิเคชันที่พัฒนาด้วย Streamlit (Python) เพื่อพยากรณ์ข้อมูล 2 ชุด ได้แก่:
1. Loan Approval Prediction (สินเชื่อ): พยากรณ์การอนุมัติสินเชื่อจากข้อมูลจำลอง
2. Titanic Survival Prediction (เรือไททานิค): พยากรณ์ผู้รอดชีวิตจากข้อมูลจริงบน Kaggle

โดยระบบจะทำการเรียนรู้ผ่านโมเดล 2 ประเภทตามข้อกำหนด:
* Ensemble Model: (Random Forest + Gradient Boosting + Logistic Regression)
* Neural Network: โครงข่ายประสาทเทียมแบบ 2 Hidden Layers

2. โครงสร้างไฟล์และโฟลเดอร์ (Directory Structure)
ก่อนเริ่มใช้งาน กรุณาตรวจสอบให้แน่ใจว่าคุณได้จัดเรียงโฟลเดอร์ดังนี้:

IS_Project/
│
├── data/                         # โฟลเดอร์เก็บข้อมูล
│   └── titanic.csv               # ต้องดาวน์โหลดจาก Kaggle มาใส่ไว้ที่นี่
│   └── dirty_loan_data.csv       # (ไฟล์นี้ระบบจะสร้างให้เองอัตโนมัติ)
│
├── models/                             # 📁 โฟลเดอร์สำหรับเก็บโมเดลที่เทรนเสร็จแล้ว
│   ├── scaler.pkl                      # - ไฟล์เก็บตัวปรับสเกลข้อมูล (StandardScaler)
│   ├── ensemble_model.pkl              # - ไฟล์โมเดล Machine Learning 
│   └── nn_model.h5                     # - ไฟล์โมเดล Neural Network
│
├── train_models.py               # โค้ดสำหรับทำความสะอาดข้อมูลและเทรนโมเดล
├── app.py                        # โค้ดหลักสำหรับสร้างหน้าเว็บ (Streamlit)
└── requirements.txt              # ไฟล์ระบุรายการไลบรารีที่ต้องใช้

3. การเตรียมความพร้อม (Prerequisites)
ก่อนรันระบบบนเครื่องคอมพิวเตอร์ของคุณ ต้องทำการติดตั้งไลบรารีที่จำเป็นก่อน:
1. เปิดหน้าต่าง Command Prompt (Windows) หรือ Terminal (Mac)
2. พิมพ์คำสั่งเพื่อเข้าไปยังโฟลเดอร์โปรเจค (เช่น cd path/to/IS_Project)
3. รันคำสั่งติดตั้งไลบรารี:
   pip install -r requirements.txt
   (หรือพิมพ์ตรงๆ: pip install pandas numpy scikit-learn tensorflow streamlit joblib)

---

4. ขั้นตอนการรันโปรแกรม (Step-by-Step Guide)

ขั้นตอนที่ 1: การสร้างโมเดล (Model Training)
วัตถุประสงค์: เพื่อให้ระบบทำการ Clean Data และเรียนรู้ (Train) เพื่อสร้างไฟล์ .pkl และ .h5
1. นำไฟล์ titanic.csv ไปวางในโฟลเดอร์ data/
2. เปิด Terminal และรันคำสั่ง:
   python train_models.py
3. รอจนระบบขึ้นข้อความว่า "กระบวนการทั้งหมดเสร็จสมบูรณ์!" (เข้าไปเช็คในโฟลเดอร์ models/ จะต้องมีไฟล์โมเดล 6 ไฟล์ปรากฏขึ้นมา)

ขั้นตอนที่ 2: การเปิดหน้าเว็บไซต์ (Run Web Application)
วัตถุประสงค์: เพื่อเปิดใช้งานหน้า User Interface ผ่านเบราว์เซอร์
1. ใน Terminal ตัวเดิม ให้พิมพ์คำสั่ง:
   streamlit run app.py
2. ระบบจะเปิดหน้าเว็บขึ้นมาบนเบราว์เซอร์โดยอัตโนมัติ (ปกติจะอยู่ที่ http://localhost:8501)

---

5. วิธีการใช้งานเว็บไซต์ (User Guide)
เมื่อเข้าสู่หน้าเว็บ คุณสามารถใช้งานเมนูด้านซ้ายมือ (Sidebar) ได้ดังนี้:

* หน้าหลักโปรเจค (Home): ใช้สำหรับดูภาพรวมของชุดข้อมูล แหล่งอ้างอิง และสถาปัตยกรรมของโมเดล
* เลือกชุดข้อมูล: (จะปรากฏเมื่อเลือกหน้าทฤษฎีหรือทดสอบ) เพื่อสลับการทำงานระหว่างข้อมูล Loan หรือ Titanic
* หน้าทฤษฎี (1 และ 3): อ่านคำอธิบายขั้นตอนการทำ Data Preprocessing (เช่น การเติมค่าว่าง) และทฤษฎีของอัลกอริทึมที่ใช้
* หน้าทดสอบ (2 และ 4): กรอกข้อมูลลงในฟอร์ม (เช่น อายุ, รายได้, เพศ) แล้วกดปุ่ม "ทำนายผล" เพื่อดูว่าโมเดลประมวลผลออกมาอย่างไร
